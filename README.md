标注  
每次得到一个没有处理的信鸽描述图片和它的相关信息，使用鼠标将图片中的虹膜位置框选出来。随后把右边输入框中的信息填好或者检查。在页面的下方是此信鸽的原页面，可以在原页面中得到更多的信息。如果可以的话，找出它的父母家庭成员的环号填上。点击下一个会把数据传到后端。
